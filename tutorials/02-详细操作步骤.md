# 📝 详细操作步骤

> 每个步骤的详细说明和注意事项

---

## 🔍 步骤1：获取职位链接

### 1.1 去招聘网站找职位

**推荐网站（从容易到难）：**

| 网站 | 难度 | 说明 |
|------|------|------|
| McKinsey.com | ⭐ | 结构清晰，AI准确 |
| BCG.com | ⭐ | 同上 |
| Bain.com | ⭐ | 同上 |
| GoldmanSachs.com | ⭐ | 同上 |
| JPMorgan.com | ⭐ | 同上 |
| LinkedIn | ⭐⭐⭐ | 需要登录，**不推荐** |

**建议：先从官网开始！**

### 1.2 找到想要的职位

```
方法A：通过搜索
↓
输入职位关键词（如 "Associate"）
↓
选择合适的职位
↓
进入职位详情页

方法B：通过分类
↓
浏览 "Careers" 或 "Jobs" 页面
↓
按部门/地点筛选
↓
点击感兴趣的职位
```

### 1.3 复制链接

**方法1：右键复制**
```
1. 在职位页面
2. 右键点击地址栏
3. 选择 "复制"
```

**方法2：快捷键**
```
Windows: Ctrl + L 选中地址，然后 Ctrl + C 复制
Mac: Command + L 选中地址，然后 Command + C 复制
```

**✅ 确认链接完整：**
```
✅ 正确：https://www.mckinsey.com/careers/associate-2026/jobs/12345
❌ 错误：https://www.mckinsey.com/careers （这是列表页，不是职位页）
```

## 🌐 步骤2：打开抓取工具

### 2.1 打开网站

```
方法1：点击收藏
↓
找到 "职位抓取工具" 书签
↓
点击打开

方法2：手动输入
↓
在地址栏输入：job-scraper.your-domain.com
↓
按回车
```

### 2.2 看到界面

你应该看到：
```
┌─────────────────────────────────────┐
│  Event Horizon Lab                  │
│  职位抓取工具                        │
├─────────────────────────────────────┤
│                                     │
│  📌 步骤1：输入职位链接              │
│  [输入框]                           │
│  [🔍 开始抓取]                       │
│                                     │
└─────────────────────────────────────┘
```

---

## 📋 步骤3：粘贴链接并抓取

### 3.1 粘贴链接

**方法1：右键粘贴**
```
1. 点击输入框
2. 右键点击
3. 选择 "粘贴"
```

**方法2：快捷键**
```
Windows: Ctrl + V
Mac: Command + V
```

### 3.2 检查链接

粘贴后，确认：
- ✅ 链接完整
- ✅ 以 http:// 或 https:// 开头
- ✅ 没有多余的空格

### 3.3 点击抓取

```
点击 "🔍 开始抓取" 按钮
↓
按钮变成 "⏳ 抓取中..."
↓
等待10-15秒
↓
自动跳转到结果页面
```

**⚠️ 注意：**
- 不要关闭网页
- 不要刷新页面
- 耐心等待10秒

---

## ✅ 步骤4：检查提取结果

### 4.1 理解界面

抓取成功后，你会看到：
```
┌─────────────────────────────────────┐
│  ✅ 抓取成功！请确认信息             │
├─────────────────────────────────────┤
│  AI信心评分：98%                    │ ← 准确度
│                                     │
│  [表单字段]                         │ ← 需要检查
│                                     │
│  [✅ 确认写入] [🔄 重新抓取] [❌ 取消]│
└─────────────────────────────────────┘
```

### 4.2 检查所有字段

**必填字段（必须有）：**

| 字段 | 说明 | 如何检查 |
|------|------|---------|
| 公司 | 完整名称 | 对不对？ |
| 职位 | 完整名称 | 有没有漏词？ |
| 行业 | 分类 | 对不对？ |
| 地点 | 城市+国家 | 准确吗？ |
| 类型 | Full-time等 | 对吗？ |
| 学历 | 要求 | 对吗？ |
| **签证** | **Yes/No** | ⭐ **重点！** |
| **日期** | **YYYY-MM-DD** | ⭐ **重点！** |
| 链接 | 申请URL | 能点开？ |

**可选字段：**
- 薪资：很多网页不写，空着没关系
- 描述：AI会自动提取

### 4.3 重点检查2个字段

**⭐ 1. 签证赞助**

AI显示的可能结果：
```
✅ Yes - 网页提到了签证/工作许可
❌ No - 网页明确说不提供
🤔 Case by case - 视情况而定
❓ Not mentioned - 网页没写

❓ 如果是 "Not mentioned"：
→ 去原网页搜索关键词 "visa", "sponsorship"
→ 找到了就改成 "Yes" 或 "No"
→ 实在找不到就保留 "Not mentioned"
```

**⭐ 2. 截止日期**

AI显示的可能格式：
```
✅ 2025-10-31 - 正确格式
✅ ASAP - 尽快
✅ Rolling - 滚动录取（招满即止）

❌ 如果日期明显错误：
→ 手动修改成正确格式
→ 或改成 "ASAP"
```

## ✏️ 步骤5：修改错误信息（如果需要）

### 5.1 如何修改字段

**所有字段都可以点击修改：**

```
1. 点击想要修改的字段
2. 删除错误内容
3. 输入正确内容
4. 点击其他地方保存
```

**示例：修改签证赞助**
```
原显示：Not mentioned
↓
点击下拉框
↓
选择：Yes
↓
自动保存
```

### 5.2 常见修改

| 情况 | 原显示 | 改成 |
|------|--------|------|
| 网页写了提供签证 | Not mentioned | Yes |
| 日期格式不对 | 10/31/2025 | 2025-10-31 |
| 城市名不完整 | London | London, United Kingdom |
| 职位名称太长 | Associate 2026 - Consulting... | Associate 2026 |

## 💾 步骤6：确认写入

### 6.1 最后检查

在点击"确认写入"前，快速过一遍：

```
□ 公司名称对吗？
□ 职位名称对吗？
□ ⭐ 签证赞助确认了吗？
□ ⭐ 截止日期对吗？
□ 工作地点准确吗？
□ 其他字段没问题？
```

### 6.2 点击写入

```
点击 "✅ 确认写入" 按钮
↓
按钮变成 "⏳ 保存中..."
↓
等待2-3秒
↓
看到成功页面
```

### 6.3 成功页面

```
┌─────────────────────────────────────┐
│                                     │
│        🎉 成功！                    │
│                                     │
│  职位已添加到第 47 行                │
│                                     │
│     [继续抓取下一个职位]             │
│                                     │
└─────────────────────────────────────┘
```

### 6.4 如果失败

**可能看到：**
```
❌ 错误：该职位已存在（第 25 行）
↓
说明：Google表格里已经有了
↓
解决：
→ 去表格第25行确认
→ 如果重复就不用管了
→ 点击"继续抓取"
```

## 🔄 步骤7：继续抓取

### 7.1 点击继续

```
点击 "继续抓取下一个职位"
↓
回到首页
↓
重复步骤3-6
```

### 7.2 批量抓取建议

**方法A：一个一个来（推荐新手）**
```
抓完1个 → 确认 → 抓下1个
✅ 优点：每个都检查，不容易出错
❌ 缺点：稍慢
```

**方法B：快速模式（熟练后）**
```
打开多个浏览器标签页
↓
每个标签页抓1个职位
↓
轮番确认
✅ 优点：快3倍
❌ 缺点：需要一定熟练度
```
