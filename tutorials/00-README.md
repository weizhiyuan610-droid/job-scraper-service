## ⚡ 3步快速指南

### 步骤1️⃣：复制职位链接

去招聘网站（McKinsey、Goldman Sachs等），找到想要的职位，复制网址

```
https://www.mckinsey.com/careers/associate-2026
```

### 步骤2️⃣：粘贴并抓取

1. 打开抓取工具网站
2. 粘贴链接到输入框
3. 点击 "🔍 开始抓取"
4. 等10秒钟

### 步骤3️⃣：检查并确认

AI会自动填写所有信息，你只需要：

**⭐ 重点检查这2个字段：**
1. **签证赞助**：Yes/No？
2. **截止日期**：格式对吗？

确认无误后，点击 "✅ 确认写入"

**完成！** 🎉

---

## 💡 记住这3点

1. **收藏网址** - 每天都要用
2. **重点检查签证和日期** - AI偶尔会错
3. **不确定就跳过** - 实在抓不了的就去掉
