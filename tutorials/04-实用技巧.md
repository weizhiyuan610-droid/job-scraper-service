# 💡 实用技巧与最佳实践

## 🎯 技巧1：批量抓取工作流

### 方法A：顺序抓取（推荐新手）

**流程：**
```
1. 准备10-20个职位链接
   ↓
2. 保存到记事本（按行）
   ↓
3. 复制第1个链接 → 抓取 → 确认
   ↓
4. 复制第2个链接 → 抓取 → 确认
   ↓
5. 重复直到完成
```

**优点：**
- ✅ 不容易出错
- ✅ 每个都检查
- ✅ 心态轻松

**缺点：**
- ❌ 稍慢（1个/分钟）

---

### 方法B：多标签页（熟练后）

**流程：**
```
1. 打开3-5个浏览器标签页
   ↓
2. 每个标签页打开抓取工具
   ↓
3. 标签1抓取职位A → 等待 → 切换标签
   ↓
4. 标签2抓取职位B → 等待 → 切换标签
   ↓
5. 轮番操作，确认时快速切换
```

**优点：**
- ✅ 效率高（3倍）
- ✅ 利用等待时间

**缺点：**
- ❌ 需要一定熟练度
- ❌ 容易手忙脚乱

**建议：熟练后再尝试**

---

### 方法C：快速清单法

**准备一个清单：**
```
今日待抓取（10个）：

☐ McKinsey - Associate 2026 - London
☐ BCG - Consultant - New York
☐ Goldman Sachs - Analyst - Hong Kong
☐ Bain - Consultant - Singapore
☐ ...
```

**操作：**
```
1. 打开清单
2. 逐个搜索职位
3. 复制链接
4. 粘贴抓取
5. 完成打勾 ☑️
```

**优点：**
- ✅ 有条理
- ✅ 不遗漏
- ✅ 可追踪进度

## ⚡ 技巧2：提高速度

### 键盘快捷键

**记住这些快捷键：**

| 操作 | Windows | Mac |
|------|---------|-----|
| 复制 | Ctrl+C | Command+C |
| 粘贴 | Ctrl+V | Command+V |
| 选中地址栏 | Ctrl+L | Command+L |
| 刷新页面 | F5 | Command+R |
| 新标签页 | Ctrl+T | Command+T |
| 切换标签 | Ctrl+Tab | Command+Tab |

**练熟这些，速度提升30%！**

---

### 快速检查流程

**不要逐字逐句看，快速扫描：**

```
扫描顺序（3秒搞定）：
1. 公司名：对吗？（1秒）
2. 职位名：完整吗？（1秒）
3. 签证：重点看！（1秒）
4. 日期：格式对吗？（1秒）
5. 其他：快速过一眼（2秒）
```

**总计：7秒/职位**

---

### 使用自动填充

**浏览器记住输入内容：**
```
1. 第一次输入完整信息
2. 浏览器自动记住
3. 下次自动填充
```

**适用于：**
- 重复的字段（如地点、行业）

## 🔍 技巧4：快速识别可抓取的网站

### 看URL判断

**容易抓取的URL特征：**
```
✅ https://www.mckinsey.com/careers/associate...
✅ https://jobs.greenhouse.io/mckinsey/...
✅ https://www.goldmansachs.com/careers/...
```

**难抓取的URL特征：**
```
❌ https://www.linkedin.com/jobs/... （需要登录）
❌ https://.../login （需要登录）
❌ 很短的URL（可能是列表页）
```

---

### 看网页结构

**容易抓取：**
```
✅ 有清晰的职位标题
✅ 有详细的职位描述
✅ 有明确的Requirements
✅ 页面结构简单
```

**难抓取：**
```
❌ 职位信息分散在多个页面
❌ 需要点击"查看更多"
❌ 有弹窗/登录墙
❌ 页面很复杂（很多无关内容）
```

## 📊 技巧5：数据质量管理

**每次必查：**

**发现问题：**

## 🔧 技巧8：问题预防

### 避免常见错误

**错误1：复制不完整的链接**
```
❌ 只复制了部分
✅ 选中整个URL（Ctrl+A）

检查：看链接是否以http开头
```

---

**错误2：没检查签证**
```
❌ AI说啥就是啥
✅ 每次都重点检查签证

原因：AI在这个字段容易错
```

---

**错误3：日期格式不对**
```
❌ 10/31/2025
✅ 2025-10-31

记忆：年-月-日（从小到大）
```

---

**错误4：忘记点击确认**
```
❌ 检查完就关网页
✅ 点了"确认写入"才算完

提示：看到"🎉 成功"才算
```
