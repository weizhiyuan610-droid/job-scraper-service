## ✅ 第一次操作示范

### 示例：抓取McKinsey职位

**1. 找到职位**
```
去McKinsey官网
↓
找到 "Associate 2026" 职位
↓
复制链接：https://www.mckinsey.com/careers/associate-2026
```

**2. 打开抓取工具**
```
打开收藏的网址
↓
看到输入框
↓
粘贴链接
```

**3. 点击抓取**
```
点击 "🔍 开始抓取" 按钮
↓
等待10秒（不要关网页）
↓
看到 "✅ 抓取成功"
```

**4. 检查信息**
```
看这些字段对不对：
✓ 公司名称：McKinsey & Company
✓ 职位：Associate 2026
⚠️  签证赞助：Yes ← 重点检查！
⚠️  截止日期：2025-10-31 ← 重点检查！
```

**5. 确认写入**
```
点击 "✅ 确认写入"
↓
看到 "🎉 成功！"
↓
职位已保存到Google表格
```

## ⚠️ 第一次使用注意

### 重点检查这2个字段：

**1. 签证赞助（最重要！）**
```
✅ Yes - 公司提供签证
❌ No - 不提供签证
🤔 Case by case - 视情况而定
❓ Not mentioned - 网页没写

如果显示 "Not mentioned"：
→ 去原网页确认
→ 或者标记为 "Case by case"
```

**2. 截止日期**
```
正确格式：2025-10-31
其他格式：ASAP / Rolling

如果是 "Rolling"：
→ 意思是"招满即止"
→ 可以保留这个值
```

## ❓ 第一个问题？

**如果第一次就遇到问题：**

1. **抓取失败**
   → 检查链接是否完整
   → 尝试刷新网页
   → 换个职位试试

2. **信息不对**
   → 直接点击字段修改
   → 或点击"重新抓取"

3. **不确定**
   → 跳过这个职位
   → 先抓简单的
