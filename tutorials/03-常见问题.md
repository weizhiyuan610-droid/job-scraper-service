### 3. Q: 为什么有些网站抓取失败？

### 10. Q: 抓取失败怎么办？

**A: 按顺序尝试这些方法**

**方法1：刷新重试**
```
1. 刷新网页（F5或Ctrl+R）
2. 重新粘贴链接
3. 点击抓取
```

**方法2：检查链接**
```
1. 确认链接完整
2. 在浏览器打开看能否访问
3. 确认是职位详情页（不是列表页）
```

**方法3：换个职位**
```
1. 可能这个网页结构特殊
2. 换个职位试试
3. 如果其他正常，说明这个无法抓取
```

**方法4：跳过**
```
1. 不是所有职位都能抓取
2. 失败率5-10%是正常的
3. 实在抓不了就手动录入
```

## ⚠️ 错误信息解读

### 错误1：无法访问网页

```
错误提示：Failed to scrape page
```

**原因：**
- 网络问题
- 网页加载失败
- 网站有反爬虫

**解决：**
1. 检查网络连接
2. 在浏览器打开链接确认能访问
3. 等几分钟后重试
4. 跳过这个职位

---

### 错误2：AI提取失败

```
错误提示：Failed to extract data
```

**原因：**
- 网页内容太少
- 网页结构特殊
- AI无法识别

**解决：**
1. 点击"重新抓取"
2. 如果还是失败，跳过这个职位
3. 手动录入

---

### 错误3：无法写入Google Sheets

```
错误提示：Failed to save
```

**原因：**
- 网络问题
- Google表格权限问题

**解决：**
1. 检查网络连接
2. 刷新页面重试
3. 联系技术支持

---

### 错误4：职位已存在

```
错误提示：This job already exists in row 25
```

**这不是错误！**

**说明：**
- Google表格里已经有这个职位了
- 可能是你之前抓的，也可能是别人抓的

**处理：**
1. 去表格第25行确认
2. 确实重复就不用管了
3. 继续抓下一个

## 🆘 紧急问题处理

### 问题：网页卡住了

**解决方法：**
```
1. 等待30秒（可能只是慢）
2. 刷新网页（F5）
3. 重新操作
4. 还是不行就关闭浏览器重开
```

---

### 问题：看不到"确认写入"按钮

**解决方法：**
```
1. 滚动到页面底部
2. 按F11进入全屏（如果屏幕太小）
3. 换个浏览器试试（推荐Chrome）
```

---

### 问题：AI信心评分很低（<70%）

**说明：**
- AI不太确定提取的内容
- 可能有错误

**处理：**
```
1. 仔细检查所有字段
2. 重点检查签证和日期
3. 不确定的字段就去原网页确认
4. 实在不行就跳过这个职位
```
